Image

An image is a graphic representation of something. Images can be made responsive so that they don't jump out of its parent container, can have rounded corners or can look like a circle.

Responsive Images

Images that work well on devices with widely differing screen sizes, resolutions.

Rounded Image

The .circle-img class shapes the image to a circle.

Rounded Corners

The .rounded-img class adds rounded corners to an image.
