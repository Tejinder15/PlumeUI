Avatar

An Avatar is a representation of users profile picture used primarily in chat and entertainment web sites to give them recognisable visual identity.

Profile Avatars

Profile Avatars can be created using the class '.avatar' inside the img tag.

Sizes

Want more control over your avatars? Add .avatar-lg, .avatar-md, .avatar-sm for additional sizes.
